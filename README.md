# WhereIsMyPi
A simple tool to find my raspberry pi on my LAN

## Usage
```
> ./whereismypi
192.168.1.116 is UP
```

## Details
My raspberry pi has listen ssh server on port 22 so I just try to connect to it using that port in my LAN, that's all!
